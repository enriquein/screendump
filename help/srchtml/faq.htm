bScreendumped FAQ

Question:
I just installed and ran the program... where is it???
Answer:
This program doesn't have much of an user interface other than the system tray icon. Look for the program's icon near the clock thingie on the bottom right corner of your screen.

If you have no clue of how it looks like, check the screenshots section.

Question:
When I run the program I get the following error: "Runtime error 48. File not found: gdiplus". What happened?
Answer:
Don't panic. This just means that the program can't find the GDI+ Dll. To fix this, download and install the Non-WindowsXP version of bScreenDumped found in the downloads area. Remember to uninstall any previous versions before reinstalling. Also remember that if you intend to run bScreenDumped from a usb flash drive (or any other kind of removable media) you should get the installer portable version instead.

Question:
A new version of bScreenDumped came out. Do I just install it over the old one?
Answer:
No. This is a bad idea. Please uninstall the previous version before installing the new one. Also remember to exit the program before attempting the uninstall.
